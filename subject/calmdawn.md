# Calm Dawn

Calm Dawn is a tech demo showing the ability of creating custom graphic effects in ejoy2d game engine.

FEATURES

* Seamless day-night-cycle
* 2D ocean simulation
* screen rotation support
* 1.1MB only

CONTROLS

* Slide left and right to control time of day
